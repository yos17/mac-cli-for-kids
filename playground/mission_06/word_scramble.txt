ANSWERS:
1. SORT    — sorts lines of text alphabetically or numerically
2. UNIQ    — removes duplicate lines (use after sort!)
3. GREP    — searches for text patterns in files
4. AWK     — a powerful text processing tool
5. CAT     — reads and displays file contents
6. LS      — lists files in a directory
7. CD      — changes your current directory
8. PIPE    — the | symbol, connects commands together
9. PWD     — prints your current working directory
