Mission 08 bulk rename lab

Goal:
  Rename 100 messy files into clean names instantly.

Steps:
  1. bash create_100_files_starter.sh
  2. ls ~/rename_100_lab | head
  3. bash rename_100_files_starter.sh
  4. ls ~/rename_100_lab | head

Safe cleanup:
  rm -ri ~/rename_100_lab

The scripts use a folder in your home directory so the mission playground
stays clean while you practice.
