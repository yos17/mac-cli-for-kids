THE SECURITY TOOLKIT:

  chmod 600 file             — only you can read/write (private!)

  chmod +x script.sh         — add execute permission

  echo "text" | base64       — encode text (NOT encryption — anyone can decode)

  openssl enc -aes-256-cbc -pbkdf2 -d -in file.enc -out file  — decrypt
