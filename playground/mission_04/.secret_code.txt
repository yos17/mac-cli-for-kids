THE READING AND WRITING COMMANDS:
  cat file.txt           — display entire file
  cat -n file.txt        — display with line numbers
  less file.txt          — scroll through large files (q to quit)
  head -5 file.txt       — first 5 lines only
  tail -10 file.txt      — last 10 lines only
  echo "text" > file.txt   — WRITE to file (replaces existing content!)
  echo "text" >> file.txt  — APPEND to file (adds to the end, safe!)
  echo "encoded" | base64 -d  — decode base64

Remember: > overwrites. >> appends. Know the difference!
