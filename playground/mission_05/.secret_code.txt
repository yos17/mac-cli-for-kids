GREP IS A DETECTIVE'S BEST FRIEND:

  grep -i "word" file.txt        — case-insensitive (finds WORD, Word, word)

  grep -l "word" *.txt           — show only filenames, not lines
  grep -n "word" file.txt        — show line numbers
  grep -c "word" file.txt        — count matching lines
